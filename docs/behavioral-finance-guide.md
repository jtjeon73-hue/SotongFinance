# 행동재무(편향) 가이드

관련: [investment-policy-guide.md](investment-policy-guide.md) · [financial-crisis-runbook.md](financial-crisis-runbook.md) · [expert-finance-standard.md](expert-finance-standard.md)

편향을 “고치면 수익이 난다”가 아니라, **상황 → 생각 → 위험 → 질문 → IPS 규칙**으로 끊는 교육용 프레임입니다.

## 사용법

각 편향마다:

1. **상황** — 언제 자주 나오는지  
2. **생각** — 머릿속 문장  
3. **위험** — 재무적으로 깨지는 지점  
4. **질문** — 매매 전 스스로/배우자에게  
5. **IPS 연결** — 미리 적어 둘 규칙

## 편향 카드 (요약)

### 손실회피·처분효과
- 상황: 손실 종목은 붙잡고 이익만 조기 실현  
- 생각: “본전만 오면 판다”  
- 위험: 포트 왜곡, 세금·기회비용  
- 질문: 지금 현금이 있다면 **같은 비중으로 다시 살까?**  
- IPS: 리밸런싱·손절이 아니라 **역할·한도** 기준 재평가

### 과신·확증편향
- 상황: 소수 정보·단톡만으로 확신  
- 생각: “나는 이미 안다”  
- 위험: 집중·사기·공시 미확인  
- 질문: **반대 증거** 3가지는? 공식·공시 봤나?  
- IPS: 금지 목록(레버리지·미확인 팁), 점검 체크리스트

### 군중·FOMO
- 상황: 급등·미디어·지인 수익 자랑  
- 생각: “지금 안 사면 바보”  
- 위험: 고점 추격, 생활비 투자  
- 질문: IPS 목적·기간에 **맞는 매수인가?** 비상자금 건드리나?  
- IPS: “급등만으로 매수 금지”, 냉각 기간(예: 24~72시간)

### 기준점·심적회계
- 상황: 매수가·어제 가격에 집착, ‘보너스 돈’은 별도 취급  
- 생각: “이 가격 밑이면 손해” / “ ind이니까 괜찮다”  
- 위험: 비합리적 보유·과소비·과투자  
- 질문: **현재** 목적·위험 한도에 맞는가? (매수가 무시)  
- IPS: 평가 기준일·역할 표, 소득 원천 구분 없이 통합 CF

### 현재편향·계획오류
- 상황: 저축·상환·서류를 미룸  
- 생각: “다음 달에”  
- 위험: 이자·연체·신고 기한  
- 질문: **오늘 15분**에 할 한 단계는?  
- IPS: 자동이체·캘린더·연간 재검토일

### 권위·친숙 편향 (사기 취약)
- 상황: 공무원·금융권 사칭, 익숙한 앱 화면  
- 생각: “공식처럼 보이니 믿어도 된다”  
- 위험: 이체·원격·개인정보 유출  
- 질문: **내가 먼저 건 공식번호**인가? ([elderly-finance-safety-guide.md](elderly-finance-safety-guide.md))  
- IPS: 금융 지시 전화 대응 규칙, 신고 경로

## 위기 시

급락·패닉에서는 편향이 겹칩니다.  
→ [financial-crisis-runbook.md](financial-crisis-runbook.md)의 **이벤트별 응답**을 IPS와 함께 사용.

## 체크리스트

- [ ] 자주 겪는 편향 2~3개만 IPS에 규칙으로 적음
- [ ] 질문 카드가 매매·이체 **직전**에 있음
- [ ] “편향만 알면 수익” 문구 없음
- [ ] 사기·노인 금융은 별도 안전 가이드와 연결
