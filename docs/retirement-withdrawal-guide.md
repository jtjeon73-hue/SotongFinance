# 은퇴 인출 가이드

관련: [pension-retirement-guide.md](pension-retirement-guide.md) · [investment-policy-guide.md](investment-policy-guide.md) · [tax-preparation-guide.md](tax-preparation-guide.md)  
콘텐츠: `pension-withdrawal`, `pension-gap`, `pension-longevity`, `pension-overview`

**만인 공통 안전인출률(SWR)을 정답으로 제시하지 않습니다.**  
“4% 규칙” 등은 특정 가정·시대의 연구 예시일 뿐, 개인 정답이 아닙니다.

## 1. 기간(구간)

| 구간 | 초점 |
|------|------|
| 은퇴 전 | 저축률, 부채, 개시 연령, 공백 자금 |
| 전환기 | 연금 개시 시점, 건보·세금, 근로 축소 |
| 초기 은퇴 | 필수비 충당, 순서 위험(초반 하락+인출) |
| 중·후기 | 장수, 의료·간병, 배우자 생존, 유동성 |

구간마다 인출·근로·지출 조정의 여지가 다릅니다.

## 2. 지출 유형

| 유형 | 예 | 인출과의 관계 |
|------|-----|----------------|
| 필수 | 주거·식비·기본 의료·최소 상환 | 가능한 한 확정소득(공적·일부 연금)과 맞춤 |
| 준필수 | 교통·통신·보험료 | 삭감 시나리오 준비 |
| 선택 | 여행·여가 | 시장·건강에 따라 조절 |
| 비정기 | 수리·간병·가족 지원 | 별도 완충(투자 인출과 분리 권고) |

월갭 ≈ 필요생활비 − (공적+퇴직+개인 등 월 확정·준확정 합) — **교육용**  
공식 수령액은 NPS 등 **공식 계산기** 권장.

## 3. 인출 방법 비교 (추천 없음)

| 방법(개념) | 장점(교육) | 리스크·조건 |
|------------|------------|-------------|
| 정액/정률 | 단순, 예산 잡기 쉬움 | 하락장·인플레에 취약할 수 있음 |
| 가변형 | 시장·지출에 맞춤 | 규칙이 없으면 감정 인출 |
| 버킷(단기현금+장기) | 순서 위험 완화에 도움될 수 있음 | 버킷 간 이전·비용·세금 설계 필요 |
| 연금화·종신형 일부 | 장수 위험 이전 측면 | 유동성·상속·비용·계약 조건 |
| 근로·사업 병행 | 인출 압력 감소 | 건강·고용 불확실 |

비교 축: **지속 가능성 · 유동성 · 세금 · 수수료 · 배우자 · 인플레**.  
개인 최적을 앱이 고르지 않습니다.

## 4. SWR을 쓰지 않는 이유 (요지)

- 수익률·인플레·수명·통화·세금·수수료 가정이 개인마다 다름  
- 초반 낙폭(순서 위험)에 따라 동일 인출률도 결과가 갈림  
- 한국 공적·퇴직·세제 구조와 해외 연구 가정이 다를 수 있음  

대신: 필수비 커버리지, 시나리오 2개 이상, 하락장 축소 규칙, 전문가·공식 도구.
