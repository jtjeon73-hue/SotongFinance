# 콘텐츠 로드맵

## 1단계 (완료 목표)

- 입문·기초 중심 학습 페이지
- 교육용 계산기 14종
- 가상 재무 시나리오 10종
- 용어·공식출처·프롬프트 라이브러리
- 검색·필터·반응형 셸

## 2단계 (실무 심화)

- 실무 난이도 페이지 확장 (세금 신고 흐름, 대출 상환 비교 심화)
- 체크리스트·워크시트 PDF/인쇄 친화
- 출처 `reviewDueAt` 만료 알림
- 생애주기별 학습 경로 큐레이션

## 3단계 (전문가 심화)

- 전문가 난이도·사례 연구
- 제도 변경 이력 타임라인
- (별도 보안·개인정보 영향평가 후) 선택적 개인 재무기록
- 접근성·다국어 검토

## 난이도 필드

`intro` → `basic` → `practical` → `expert` 확장 가능하도록 모델에 포함.
