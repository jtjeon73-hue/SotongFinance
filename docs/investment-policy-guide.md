# 투자정책서(IPS) 가이드

관련: [expert-finance-standard.md](expert-finance-standard.md) · [investment-risk-guide.md](investment-risk-guide.md) · [asset-allocation-risk-guide.md](asset-allocation-risk-guide.md) · [behavioral-finance-guide.md](behavioral-finance-guide.md)

IPS는 **수익을 보장하는 문서가 아니라**, 목적·한도·규칙을 적어 **일관된 판단을 돕는** 교육용 템플릿입니다.  
상품·종목 추천 없음. 기본 배분 비율(예: 주식 60%)을 사이트 정답으로 두지 않습니다.

## IPS가 하는 일 / 하지 않는 일

| 함 | 안 함 |
|----|------|
| 목적·기간·유동성·손실 한도 명문화 | 수익률·원금 보장 |
| 위기 시에도 지킬 규칙 사전 합의 | 만인 동일 자산배분 |
| 행동편향 대응 질문과 연결 | 매매 시점·종목 지시 |

## 템플릿 필드 (복사해 로컬에만)

| 필드 | 적을 내용 |
|------|-----------|
| 작성일·재검토일 | `reviewDueAt`과 맞춤 |
| 목적 | 은퇴·주택·교육·비상 등 (복수 가능, 우선순위) |
| 투자기간 | 목적별 연수·인출 시작 예상 |
| 필수 현금·비상 | 투자 비중 **밖**에서 확보할 금액/개월 |
| 손실 한도 | “이 이상 하락 시 생활·수면에 지장” 수준(성향≠능력) |
| 허용 자산 유형 | 예금·채권·주식·ETF·부동산 등 **역할**만 (비중 정답 없음) |
| 금지·제한 | 레버리지, 신용매수, 생활비 투자 등 |
| 리밸런싱 규칙 | 주기·밴드·세금·비용 고려 ([asset-allocation-risk-guide.md](asset-allocation-risk-guide.md)) |
| 인출·납입 규칙 | 정액/변동, 하락장 축소 여부 |
| 의사결정 금지 상황 | 급등·급락·단톡 팁만으로 매매 금지 등 |
| 가족·대리 | 배우자 합의, 위임 범위 (자율 존중 → [elderly-finance-safety-guide.md](elderly-finance-safety-guide.md)) |
| 전문가 의뢰 | 세무·법률·투자자문 경계 |

민감 금액·계좌번호는 서버에 올리지 않습니다 ([privacy-review.md](privacy-review.md)).

## 작성 순서 (교육용)

1. 목적·기간·필수 현금  
2. 손실 감수 **능력**(부채·필수비) → 그다음 **성향**  
3. 자산 **역할** 정의 (비율 숫자보다 먼저)  
4. 리밸런싱·금지 규칙  
5. 행동편향 질문과 연결 ([behavioral-finance-guide.md](behavioral-finance-guide.md))  
6. 연 1회 또는 인생 사건 시 재검토

## 하지 말 것

- “주식 N% / 채권 M%가 정답”처럼 기본 배분 제시
- IPS만 있으면 손실이 없다는 암시
- 타인 IPS를 그대로 복사해 내 정답으로 사용

## 체크리스트

- [ ] 목적·기간·비상자금이 비율보다 위에 있음
- [ ] 손실 한도 = 성향과 능력 구분
- [ ] 고정 배분·수익 보장 문구 없음
- [ ] 리밸런싱·금지·재검토일 기입
- [ ] 로컬 보관, 서버·URL 미전송
