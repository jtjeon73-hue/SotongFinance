# 계산기 검증 가이드

## 원칙

- 모든 수식은 `FinanceMath`에 두고 단위 테스트로 교차검증
- 단위(원/%/년/개월)를 UI에 명시
- 세금·연금 공식 계산기를 대체하지 않음
- 결과 복사 가능, 서버 저장 없음

## 교차검증 예

- 단리: P×(1+r×t)
- 복리: P×(1+r/n)^(n·t)
- 원리금균등: 표준 amortization 공식
- 원금균등 총이자: P×월이율×(n+1)/2
- 실질수익률: (1+명목)/(1+물가)−1

## 비정상 입력

0 소득(저축률), 0 매수금액, 0 개월, 음수 복리횟수 등은 ArgumentError로 처리합니다.
