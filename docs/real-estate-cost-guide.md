# 부동산 비용·수익 가이드

관련: [practical-finance-standard.md](practical-finance-standard.md) · [source-verification-guide.md](source-verification-guide.md)  
콘텐츠: `re-total-cost`, `re-rental-basics`, `re-buy-rent`, `re-farmland`, `re-mortgage`

특정 지역·물건 매수 추천 없음. 세금·규제는 공식자료 재확인.

## 1. 총비용 (가격 ≠ 총투입)

| 단계 | 포함 예 |
|------|---------|
| 취득 | 매매가, 계약·중도·잔금, 취득 관련 세금, 중개·등기·법무, 대출비용, 즉시 수리 |
| 보유 | 이자, 보유세, 관리·보험, 수선, 공실 |
| 처분 | 중개, 양도세 **검토**(세액 확정 금지), 이사·공백 비용 |
| 기회 | 동일 자기자본의 다른 운용 |

자기자본과 대출을 구분하고, 하락·공실·대형수선·금리상승 시나리오를 넣습니다.

계산기: 총취득비용 합계, 순임대현금흐름, 공실·수선 스트레스

## 2. 임대수익률 유형 (섞지 말 것)

| 유형 | 의미 |
|------|------|
| 표면 임대 | 연 임대료 ÷ 가격 (비용 미반영) |
| 순임대 | 공실·수선·관리·세금·이자 등 반영 후 |
| 자기자본 | 실제 넣은 자기자본 대비 |
| 매각 전/후 | 보유 중 현금 vs 매각 손익 포함 |
| 세전/세후 | 세율은 공식·전문가 확인 |

표면수익만으로 대출을 최대화하지 않습니다.

## 3. 농지 점검

`professionalReviewRequired` 주제에 가깝습니다.

- 지목·실제 이용·농지 여부·농업진흥지역
- 취득 자격·이용 의무
- 진입도로·경계·용수·배수·전기
- 건축·개발·산지·빈집 관련 제한
- 공부(登記)와 **현장** 불일치 여부
- 수요·매도 가능성(유동성)

등기부만으로 모든 제한을 확인했다고 보지 마세요. 공식·행정·전문가 상담.

## 4. 실거주 vs 투자

| | 실거주 | 투자(임대·시세) |
|--|--------|----------------|
| 주 질문 | 생활 안정·통근·교육·의료·보유기간 | 순현금·공실·세금·환금성 |
| 주의 | 투자 수익만으로 통근 불가 주택 | 단기 전매 가정과 실거주 혼동 |

가능 대출액 ≠ 생활 감당액. LTV·DSR 등은 시점·지역 의존 (`versionDependent`).
