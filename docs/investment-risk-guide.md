# 투자 위험 가이드

관련: [practical-finance-standard.md](practical-finance-standard.md) · [calculator-validation-guide.md](calculator-validation-guide.md)  
콘텐츠: `stock-basics`, `stock-portfolio-risk`, `basics-risk-return` 및 ETF 관련 페이지

매매 신호·종목 추천 없음. 과거 수익률 ≠ 미래.

## 1. 주식 학습 흐름 (분석 순서)

1. 사업·고객·수익 모델  
2. 매출·비용·이익  
3. 현금흐름·부채·자본  
4. 경쟁·성장 가정  
5. 위험 목록  
6. 주식 수·가치 기대와 가격의 차이  
7. 불확실성·검증 자료(공시)

가격만 보고 재무·공시를 건너뛰지 않습니다.

## 2. ETF 체크리스트

- [ ] 추종 지수·보유 자산 이해
- [ ] 총보수·기타 비용 장기 영향
- [ ] 추적오차·괴리(프리미엄/디스카운트)
- [ ] 거래량·유동성
- [ ] 분배(배당) 구조·세금 개념
- [ ] 환헤지 여부·통화 위험
- [ ] 실물/합성 등 구조
- [ ] 레버리지·인버스(복리·경로의존) 위험
- [ ] 상장폐지·조기청산 가능성
- [ ] 투자 목적·기간·손실 한도와 맞는지

계산기: ETF 비용 장기영향, 투자 손익률, 손실 회복률

## 3. 위험 유형 (예시)

| 유형 | 요지 |
|------|------|
| 시장 | 전체 시장 하락 |
| 개별 | 특정 기업·섹터 |
| 신용 | 부도·등급 악화 |
| 금리 | 채권가격·대출비용 |
| 환율 | 외화자산·환헤지 |
| 유동성 | 원하는 시점에 현금화 어려움 |
| 집중 | 소수 종목·지역·자산 편중 |
| 레버리지 | 손실 확대·경로의존 |
| 비용 | 보수·거래·세금이 복리로 잠식 |
| 심리·사기 | 추격매수, 원금보장·비밀정보 유혹 |

감수 **성향**과 감수 **능력**(필수비·부채·비상자금)을 구분합니다.

## 4. 자산배분 원칙 (고정 비율 없음)

- 목적·기간·유동성·손실 한도가 비율보다 앞선다
- “주식 N% / 채권 M%”를 만인 정답으로 제시하지 않음
- 비상·단기 필요자금을 투자 비중 밖에서 먼저 확보
- 리밸런싱은 규칙·비용·세금을 함께 검토
- 배분 변동은 교육용 단순 계산으로만 시연 (`calc-allocation-shift`)

## 5. 잘못된 판단 예

- 팁·단톡만 보고 공시 미확인  
- 생활비·대출로 투자  
- 레버리지 ETF를 장기 정액적립처럼 취급  
- 분산했다고 착각한 동종 집중
