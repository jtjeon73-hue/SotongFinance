# 고령자 금융 안전 가이드

관련: [behavioral-finance-guide.md](behavioral-finance-guide.md) · [privacy-review.md](privacy-review.md) · 콘텐츠 `fraud-*`, `mm-records-docs`

목표는 **자율을 존중하면서** 사기·복잡 계좌· ind치 않은 대리 통제를 줄이는 것입니다.  
가족이 임의로 계좌·인감·앱을 장악하는 것을 권장하지 않습니다.

## 1. 계좌·절차를 단순하게

- 사용 계좌·카드를 **필요한 수**로 정리 (폐지·휴면은 본인 확인 후)
- 자동이체·연금 수령 계좌를 한눈에 보이게
- 복잡 투자·레버리지·모르는 앱 설치 최소화
- 중요 서류·비밀번호는 **본인이 관리**할 수 있는 방식(기억·보관)으로

단순화 ≠ 타인에게 전권 위임.

## 2. 사기·이상 거래 신호

- 검찰·금융당국·자녀 사칭, 원격제어, “안전계좌 이체”
- 고수익·원금보장·비밀 투자  
- 급하게 현금·상품권·코인 요구  

대응: **내가 먼저 건 공식 번호**로 재확인 → 이체·원격 거부 → 금융회사·경찰·금융감독원 등 신고 경로 ([fraud 콘텐츠](content-roadmap.md)).

공포를 과장하거나 “고령이면 무조건 당한다”고 쓰지 않습니다.

## 3. 자율 존중 · 가족 역할

| 권장 | 비권장 |
|------|--------|
| 본인 의사 확인 후 설명·동행 | 동의 없는 계좌 이관·비밀번호 갈취 |
| 위임·후견은 **법적 절차**로 | “가족이니까” 임의 인출·투자 |
| 큰 이체는 냉각 시간·이중 확인 | 공포로 모든 결정을 가로챔 |
| 기록(언제·무엇을 설명했는지) | 수치심을 주는 훈계만 |

의사결정 능력이 우려되면 의료·법률·후견 제도를 **전문가와** 검토합니다. 앱이 후견을 대체하지 않습니다.

## 4. IPS·기록과의 연결

- 대리 범위·금지 이체·신고 규칙을 IPS에 짧게 ([investment-policy-guide.md](investment-policy-guide.md))
- 민감 정보는 서버에 올리지 않음
