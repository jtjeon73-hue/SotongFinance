# 연금·은퇴 현금흐름 가이드

관련: [practical-finance-standard.md](practical-finance-standard.md) · [source-verification-guide.md](source-verification-guide.md)  
콘텐츠: `pension-overview`, `pension-nps-*`, `pension-retire-account`, `pension-personal`, `pension-gap`, `pension-withdrawal`, `pension-longevity`

개인 수령액·최적 인출률을 단정하지 않습니다. 국민연금 등은 **공식 계산기** 권장.

## 1. 4층 구조 (교육용 구분)

| 층 | 예 | 확인할 것 |
|----|----|-----------|
| 1 | 국민연금 등 공적 | 가입기간, 예상 수령(공식), 개시 연령 |
| 2 | 퇴직연금(DB/DC/IRP 등) | 제도·잔액·수수료·수령 방법(계약·안내) |
| 3 | 개인연금·저축성 | 세제·비용·유동성(`versionDependent`) |
| 4 | 근로·사업·임대·금융자산 인출 | 세금·시장·건강·노동 가능 여부 |

층마다 위험·세금·유동성이 다릅니다. “연금만으로 충분”을 전제하지 마세요.

## 2. 은퇴 현금흐름 구간

1. **은퇴 전 적립** — 저축률, 투자가능액, 부채 정리  
2. **은퇴 전환** — 개시 시점, 공백 기간, 건강보험·세금  
3. **초기 은퇴** — 필수비 vs 연금 합, 비상·의료 완충  
4. **후기·장수** — 수명 연장, 간병, 인출 지속 가능성  

월갭 ≈ 필요생활비 − (공적+퇴직+개인 등 월 수령 합) — 교육용  
계산기: 연금 월수령 합계, 은퇴자금 부족액, 은퇴기간 월갭, 물가 미래생활비

## 3. 주요 위험

- 장수(longevity): 기간이 길어질수록 고갈·의료비
- 수익률 **순서 위험**: 은퇴 초 하락 + 인출
- 인플레이션: 실질 생활비 상승
- 금리·시장: 잔액·재투자
- 제도·세제 변경 (`versionDependent`)
- 유동성: 비연금 자산의 현금화 비용
- 가족·상속: 배우자 생존·의료

## 4. 인출 — 고정 인출률 금지

“4% 규칙” 등 **만인 고정 인출률을 정답으로 제시하지 않습니다.**

대신:

- 필수비 / 선택비 구분
- 연금 확정분 vs 변동 자산 인출 구분
- 하락장 인출 축소·근로·지출 조정 시나리오
- 세금·수수료·계좌 인출 순서(계약·세무 확인)

## 5. 체크리스트

- [ ] 4층별 잔액·개시·비용 표
- [ ] 공식 NPS 등 예상액 재조회
- [ ] 은퇴 전·후 월 CF 시나리오 2개 이상
- [ ] 의료·간병·비상 완충
- [ ] `reviewDueAt` 도래 시 제도 재확인
- [ ] 필요 시 전문가 (세무·연금 상담)
