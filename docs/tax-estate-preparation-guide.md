# 세금·상속·증여 준비 가이드

관련: [tax-preparation-guide.md](tax-preparation-guide.md) · [source-freshness-guide.md](source-freshness-guide.md) · [expert-finance-standard.md](expert-finance-standard.md)  
콘텐츠: `tax-*`, `re-capital-gains`, 상속·증여 관련 페이지

**앱에서 세액을 확정하지 않습니다.** 준비자료·일정·질문 목록까지입니다.  
세율·공제·평가 방법은 `versionDependent` — 공식 안내·필요 시 **세무·법률 전문가**.

## 원칙

1. 준비 ≠ 신고 대리 ≠ 절세 상품 추천  
2. 국세청·관할 안내·공식 계산기를 우선  
3. `professionalReviewRequired` / `officialCalculatorRecommended` 존중  
4. 민감 증빙·가족 재산 목록은 로컬만 ([privacy-review.md](privacy-review.md))

## 1. 일반 세금 준비 (요약)

[tax-preparation-guide.md](tax-preparation-guide.md)와 동일:

- 소득·경비·원천징수 분류  
- 기한 캘린더  
- 공제·세액공제는 **공식에서 해당 여부 확인**  
- 양도: 양도가→취득가→필요경비→보유·거주→주택 수 등 **검토 흐름**만

## 2. 상속·증여 — 교육 범위

| 할 일 | 하지 말 일 |
|-------|------------|
| 가족·자산·부채·보험·보증 **목록** (로컬) | 블로그 세율로 납부액 확정 |
| 신고·납부 **기한·관할·서류** 공식 확인 | “미리 다 이전하면 안전” 단정 |
| 유언·후견·부동산 등기 이슈를 **질문**으로 정리 | 법률 자문 대체 |
| 평가·공제·세율은 시점 의존임을 고지 | 앱에 영구 세율표 |

증여·상속은 가족 관계·재산 형태·시점·평가에 따라 달라집니다.  
분쟁·유언 효력·등기는 **법률 전문가**, 세액·신고는 **세무 전문가** 영역으로 분리합니다.

## 3. 전문가 검토가 필요한 신호

- 부동산·비상장·해외자산·사업체  
- 배우자·자녀·손자녀 등 복수 수증자  
- 채무·보증·담보가 얽힌 경우  
- 사전증여 이력·평가 다툼 가능  
- 세무조사·기한 임박

본문은 “가져갈 서류·질문 목록”만 제공합니다.

## 4. 질문 목록 예시 (상담 전)

- [ ] 자산·부채·보험 수익자 목록 기준일  
- [ ] 취득·증여 이력·관련 계약  
- [ ] 확인한 공식 페이지 URL·확인일  
- [ ] 신고 기한까지 남은 일수  
- [ ] 가족 합의·유언·후견 관련 열린 이슈  

## 5. 메타데이터

| 필드 | 용도 |
|------|------|
| `referenceYear` / `checkedAt` / `reviewDueAt` | 세금·상증은 **짧은 재확인 주기** |
| `verificationStatus` | versionDependent / needsReview / professionalReviewRequired |

- 타인 사례 세액을 내 예상으로 복사
