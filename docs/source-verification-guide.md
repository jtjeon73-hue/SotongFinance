# 공식 출처 검증 가이드

## 상태

| 상태 | 의미 |
|------|------|
| verified | 공식 기관 랜딩/안정 페이지가 설명을 뒷받침 |
| versionDependent | 수치·한도·규제가 시점 의존 |
| needsReview | 절차·연락처 등 재확인 필요 |
| officialCalculatorRecommended | 공식 계산기 사용 권장 |
| professionalReviewRequired | 세무·법률 전문가 확인 권장 |
| educationalExample | 교육용 예시 |

URL 존재만으로 verified 처리하지 않습니다.

## 재확인

`reviewDueAt`이 지나면 UI에서 “최신 공식자료 재확인 필요”를 표시할 수 있습니다.
