# 가계 재무 가이드

관련: [practical-finance-standard.md](practical-finance-standard.md) · 콘텐츠 `mm-household-status`, `mm-emergency-fund`, `mm-debt-payoff`, `mm-financial-goals`

교육용 점검 순서입니다. 상품 추천·고정 비율 정답은 없습니다.

## 1. 대차대조표(BS)와 현금흐름(CF)

| 표 | 질문 | 핵심 식 |
|----|------|---------|
| BS | 지금 순자산은? | 순자산 = 총자산 − 총부채 |
| CF | 이번 달/해에 남는 현금은? | 현금흐름 = 소득 − (고정·변동지출 + 세금 + 보험 + 상환 + 저축) |

둘을 **함께** 봅니다. 자산이 커도 CF가 적자면 위험하고, 소득이 높아도 부채·지출로 순자산이 정체될 수 있습니다.

자산: 현금·예금·투자·부동산·사업/농업·연금성  
부채: 주담대·신용·사업·농업·카드·**보증**

기준일·평가방법(시세 vs 장부가)을 메모하세요. 민감 숫자는 로컬에만 보관합니다.

## 2. 목표 우선순위 (기본 순서, 개인차 허용)

1. 현황(BS·CF) 파악  
2. 필수 생활비 확정  
3. 비상자금 설계  
4. 고금리·연체 위험 부채  
5. 보험 공백·중복 점검  
6. 단기 필요자금  
7. 노후·연금 연결  
8. 장기 투자 후보  
9. 세금 준비  
10. 연간 재점검  

가족·자영·농민·은퇴는 3~5단계 비중이 달라질 수 있습니다.

## 3. 비상자금

- 충당개월 ≈ 현금성 자산 ÷ 월 필수지출 (교육용)
- **직업·소득 안정성**에 따라 목표를 다르게 (직장 / 자영·매출공백 / 농민·계절·재해 / 은퇴·의료·인출)
- 사업 운영자금과 생활 비상자금을 섞지 않음
- 투자계좌를 비상자금으로 쓰면 하락+인출이 겹칠 수 있음
- “무조건 N개월”을 사이트 정답으로 두지 않음

계산기: 비상자금 충당개월 (`calc-emergency-months`)

## 4. 부채

부채별 **잔액·금리·월상환·만기·수수료·담보·보증** 표로 비교합니다.

- 고금리 우선 / 작은 잔액 우선 / 현금흐름 안정 / 만기·변동금리 위험 — 전략을 **나란히** 비교
- 비상자금 없이 전액 상환만 고집하지 않음
- 대환은 수수료·조건을 공식·계약으로 확인 (`versionDependent`)

계산기: 대출 부담·총이자·금리 스트레스
