# 변동소득(자영·농민) 가이드

관련: [household-finance-guide.md](household-finance-guide.md) · [tax-preparation-guide.md](tax-preparation-guide.md)  
콘텐츠: `mm-variable-income`, `mm-emergency-fund` · 경로: 자영업자·농민 life path

평균 소득만 보면 비수기·세금월에 계획이 무너집니다.

## 1. 연간·계절 달력

12~24개월에 표시:

| 항목 | 예 |
|------|----|
| 입금·매출 집중월 | 성수기, 출하, 계약금 |
| 원가·농자재·설비 | 파종, 사료, 수리 |
| 인건·임차·고정운영비 | 매월 |
| 대출 만기·이자 | 갱신월 |
| 세금·보험료 | 신고·납부월 |
| 생활비 인출 | 개인 계좌로 |
| 비상 보충 | 재해·거래처 지연 |

월 현금 ≈ 입금 − (원가 + 인건 + 임차 + 상환 + 세금 + 생활비)  
**연간 합계**와 **최저월**을 함께 봅니다.

계산기: 변동소득 연간 합계 (`calc-variable-income`)

## 2. 개인 돈 / 사업 돈 분리

- 사업(농업) 수입 계좌와 생활비 계좌를 나눔
- 매출 ≠ 순이익 ≠ 인출 가능액
- 사업 운영자금과 가계 비상자금을 구분 ([household-finance-guide.md](household-finance-guide.md))
- 카드·대출을 생활비와 사업비로 섞지 않음
- 증빙·장부는 세금 준비와 연결 ([tax-preparation-guide.md](tax-preparation-guide.md))

## 3. 위험 (동시 충격)

재해·가격 하락 · 거래처 지연 · 금리 상승 · 세금월 · 설비 고장 · 건강·노동 공백

좋은 달 지출을 기본값으로 두지 마세요.

## 4. 체크리스트

- [ ] 12~24개월 달력 작성

- [ ] 사업 비상 ≠ 가계 비상
- [ ] 민감 장부는 로컬만
